## upi qrcode

Generate Qrcode from String

### Synopsis

Generates a qrcode and prints it as a string to the stdout. UPI request/send strings can be used as input (string format)

```
upi qrcode [flags]
```

### Examples

```
upi qrcode 'upi://pa.... '
```

### Options

```
  -h, --help   help for qrcode
```
